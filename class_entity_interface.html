Detailed Description

Stores crucial information such as (Health, Mana, Money, etc). Also provides functions that can get/set Stats. This script is attached to all Entities, including the Player. It is kept generic and all encompassing so that anything that needs to access stats, can.

Initialized by the <EntityName>Configuration.cs script attached to <EntityName>/Configuration

Attached to:

  1. <EntityName>/Interface

Inter-Node Communication:

  1. Accessed by any Entity or object that needs to modify the stats of this entity.
  2. By using Adder, Setter, and Getter functions we enable Signals; just creating properties alone would reduce flexibility.

Programmers Notes:

  1. Used by both Player and NPC's to hold their parameters and get/set things.
  2. Some getters/setters use enumerations from KoboldEnumerations namespace, which is related to EnumerationsTemplate.cs.
