Signals Explained

Godot Signal Documentation.

Signals are Godot's version of the observer pattern, allowing nodes to send out a message that other nodes can listen for and respond to. This allows for decoupling.

WARNING: You will want to turn ON contact_monitor and set contacts_reported to a value for the maximum number of contacts reported, in order to use Area2D/KinematicBody2D detections with signals.

Signals in Use

To find the signals in use for this project, do a file-contents search for .Connect keyword.

Signals Workflow

Using the SensorCcontroller.cs as an example, you must do the following to utilize Signals.

Build the Call Function

The first step is to build a function to be called once the signal we're subscribing to publishes.

public void <the_function_to_call_in_your_script>(<input_params>){
<functionality>
}

Example from SensorController.cs:

public void _on_Sight_area_entered(Area2D area){
AddEntityToAwareness(area.GetParent());
}
public void _on_Sight_area_exited(Area2D area){
RemoveEntityFromAwareness(area.GetParent());
}

Establish the Connection

The second step is to establish the connection in the _Ready() function. Once this is done, any time the signal is generated this function will be called.

public override void _Ready()
{
this.Connect("<the_function_generating_the_signal>", this, "<the_function_to_call_in_your_script");
}

Example from SensorController.cs:

public override void _Ready()
{
this.Connect("area_entered", this, "_on_Sight_area_entered");
this.Connect("area_exited", this, "_on_Sight_area_exited");
}

Custom Signals

To declare your own custom signal:

public class Main : Node2D
{
[Signal]
public delegate void MySignal();
}

once declared, the custom signal should appear in the inspector and can be connected the same way as a node's built-in signals.

public class Main : Node2D
{
[Signal]
public delegate void MySignal();
public override void _Ready()
{
EmitSignal(nameof(MySignal));
}
}

A signal can also optionally declare one or more arguments:

public class Main : Node
{
[Signal]
public delegate void MySignal(bool value, int other_value);
}

To pass values, add them as the second argument to the emit_signal function:

public class Main : Node
{
[Signal]
public delegate void MySignal(bool value, int other_value);
public override void _Ready()
{
EmitSignal(nameof(MySignal), true, 42);
}
}
