Detailed Description

Tutorial: Raycasting https://www.youtube.com/watch?v=esZdJegvANc

Base class for NPC ability to move, interact, attack, and perform other kinds of actions. Drives the Current State in the EntityInterface, which in turn drives the Animations of this character.

Attached to: NPC/Controllers

Inter-Node Communication:

  1. Works with Raycast2DFront to detect when colliding with other creatures or walls. It does this via the SensorInterface.

Programmers Notes:

  1. Interfaces with the base node using the Owner key word, which is a KinematicBody2D.
  2. Makes use of the KoboldEnumerations for the modes, to improve readability.
  3. WARNING: When the character moves left/right and flips horizontal scale, this also flips the Raycasts. That's why they are called front/rear instead of right/left.
