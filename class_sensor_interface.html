Detailed Description

Tutorial: N/A

Interface for the SensorController. Use this instead of directly referencing SensorController.cs. Anything that involves detection and sensing between entities goes through this class.

Attached to: Entity/Interfaces/SensorInterface

Inter-Node Communication:

  1. Used by the MovementController for the 2DRaycastLeft/Right flags.

Programmers Notes:

Parameters
allDetectedEntitiesA list of Nodes, set every frame by the SensorController.cs.
enemiesA subset of allDetectedEntities, filtered by whether their faction is considered 'Enemy'.
alliesA subset of allDetectedEntities, filtered by whether their faction is considered 'Ally'.
rearRaycast2DFlagBoolean set to True when something collides with the Raycast2DRear node.
frontRaycast2DFlagBoolean set to True when something collides with the Raycast2DFront node.
frontCollisionObjectThe Object detected when Raycast2DFront collides with something.
rearCollisionObjectThe Object detected when Raycast2DRear collides with something.
