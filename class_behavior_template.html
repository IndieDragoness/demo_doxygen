Detailed Description

Tutorial: N/A

Base class for NPC behaviors.

Attached to: N/A, serves as a parent class to specific behaviors.

Inter-Node Communication:

  1. N/A

Programmers Notes:

  1. When creating behaviors, inherit from this class.
  2. Only include things here that are broadly applicable.
  3. In more specific child-behaviors, use override to replace functionality. For instance, to change the way a the behavior moves the character, replace BehaviorMovementHandler().

@params behaviorActive A bool that determines if this particular behavior is active or not. @params targetLocation A Vector2 that stores the current target location. Could be the location of an enemy, or a patrol point. Used by the Mover() function.

◆ BehaviorMainLoop()

void BehaviorTemplate.BehaviorMainLoop ( )

The 'Main Loop' of a behavior. Encapsulates the process of executing a behavior, and is only active when behaviorActive is true. While behaviorActive is true, other parts of a behaviors functionality may not be active; for instance, if moverActive is false then the entity won't move.
