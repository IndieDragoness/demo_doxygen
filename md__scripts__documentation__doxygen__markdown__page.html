Currently, Doxywizard is stored at Scripts/Documentation/doxywizard. This is the file you will use to re-generate the HTML documentation.

Dark Theme and Modern Styling

It pulls styling from Scripts/Documentation/CSS/doxygen-awesome.css, which is set inside of doxywizard under the Expert/HTML/HTML_EXTRA_STYLESHEET option. For example, it's currently set to .\CSS\doxygen-awesome.css which uses a relative path within the project. Relative paths enable pushing/pulling to Git without breaking the link.

Using this Readme as the Main Page

To set README.md file as the main page for the Doxygen generated documentation, use the USE_MDFILE_AS_MAINPAGE parameter in the configuration.

Removing the Long Directory Name from Files

Currently, Doxygen references your personal path when listing where files are stored in the directory on each Class page. To fix this, leave FULL_PATH_NAMES = YES and then set STRIP_FROM_INC_PATH = src/.

Dot and GraphViz for Generating Diagrams

GraphViz Website. Doxygen GraphViz Page.

The GraphViz third party tool is used to generate diagrams. To make it available to Doxygen, install it using the executable and be sure to click 'Add to Path' during installation.
