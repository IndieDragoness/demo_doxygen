Overview

The ArtificialIntelligenceController is the highest level script, containing the 'Main' loop for AI for a given Entity. Each Entity will have it's own controller.

The BehaviorInterface ties all of the AI scripts and the SensorController together.

The BehaviorTemplate is the parent class of all behaviors. Used for commonly needed functions.

The BehaviorConfiguration

The MeleeAttackBehavior implements the entities melee attack (if it has one).

The RangedAttackBehavior implements the entities ranged attack (if it has one).

The PatrolBehavior implements patrolling.

The MoveBehavior implements 'pathfinding' to a specific location.

The IdleBehavior implements how this entity behaves while Idle.
